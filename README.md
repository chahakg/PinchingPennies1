# Introduction

## What is PinchingPennies?
  
  This project is a web app designed to help students, especially those at UIUC, manage their finances more effectively. It allows users to create monthly budgets, track daily expenses, and stay aware of their spending habits.
  The site is useful for preventing overspending and encouraging better financial choices throughout the month. By focusing on student lifestyles, it provides a practical budgeting experience compared to generic finance tools.


## Key Features:
  - Simple and intuitive UI/UX
    
  - Customizable budget categories
    
  - Easy expense tracking within each category
    
 -  Budget updates and spending insights
